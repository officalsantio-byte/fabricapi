Все о Fabric API

MinecraftClient

Чтобы получить объект клиента Minecraft используйте:

MinecraftClient mc = MinecraftClient.getInstance();

PlayerEntity

Чтобы получить игрока из клиента, используйте `mc.player`.

PlayerEntity player = mc.player;

World

Чтобы получить мир, используйте `mc.world` и затем работайте с сущностями или блоками.

World world = mc.world;

PlayerInventory

Игрок хранит предметы в своем инвентаре. Получить его можно так:

PlayerInventory inventory = mc.player.getInventory();

ItemStack

Чтобы получить предмет, который держит игрок, используйте:

ItemStack item = mc.player.getMainHandStack();

BlockState

Получение состояния блока по координатам выглядит так:

BlockState block = world.getBlockState(new BlockPos(x, y, z));

getHealth

Получение текущего здоровья игрока:

float health = mc.player.getHealth();

Координаты

Получение координат игрока по осям X, Y и Z:

double x = mc.player.getX(); double y = mc.player.getY(); double z = mc.player.getZ();

getYaw

Получение горизонтального поворота игрока:

float yaw = mc.player.getYaw();

getPitch

Получение вертикального наклона игрока:

float pitch = mc.player.getPitch();

getEntities

Получение всех сущностей из мира:

List<Entity> entities = world.getEntities();

getPlayers

Получение всех игроков из мира:

List<PlayerEntity> players = world.getPlayers();

isPressed

Проверка, нажата ли клавиша. Например, пробел, shift или sprint:

boolean jump = mc.options.jumpKey.isPressed();
boolean sneak = mc.options.sneakKey.isPressed();
boolean sprint = mc.options.sprintKey.isPressed();

Vec3d

Создание вектора и вычисление направления между точками:

Vec3d target = new Vec3d(x, y, z); Vec3d player = mc.player.getPos(); Vec3d dir = target.subtract(player).normalize(); float yaw = (float) Math.toDegrees(Math.atan2(dir.z, dir.x)) - 90; float pitch = (float) Math.toDegrees(Math.asin(dir.y));

Удар

Пример удара по сущности в радиусе 5 блоков:

for (PlayerEntity target : mc.world.getPlayers()) { if (target.distanceTo(mc.player) <= 5) { mc.interactionManager.attackEntity(mc.player, target); mc.player.swingHand(Hand.MAIN_HAND); } }

attackEntity

Атака сущности через менеджер взаимодействий Minecraft:

mc.interactionManager.attackEntity(mc.player, target);

FPS

Получение текущего FPS в клиенте:

int fps = mc.getCurrentFps();

Game Version

Получение версии игры:

String version = mc.getGameVersion();
